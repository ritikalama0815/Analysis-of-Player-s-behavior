# Analysis-of-Player's-behavior
Factors influencing spending in gaming apps are an important area of interest within the gaming industry. The project explores the connection between player behavior and the amount of in-game purchases through a gaming dataset which has variables including demographic information, time playing the game, number of sessions played, average session duration, and spending amount among other features. Data cleaning involved imputation and generation of additional engagement variables, while visualization techniques helped to establish trends and group players. There are three primary approaches used in this project; KMeans clustering analysis, Linear regression analysis, and decision trees, which include Random Forest and CART. The findings reveal that there is a strong correlation between player behavior and spending. Although the relationship is nonlinear, clustering analysis reveals distinct segments based on spending while regression models indicate that time playing, session engagement, and purchase timing predict spending. Nevertheless, predicting player behavior is challenging since the distribution of players' spending follows a skewed pattern where very few players make heavy contributions to total spending.
